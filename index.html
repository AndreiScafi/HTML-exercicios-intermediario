Formulário de HTML, comunicação e integração entre frontend e backend.

Na tag form, tudo que estiver dentro da tag seria enviado quando o formulario for submetido.

Method GET os dados são enviados no endereço, deve ser utilizado em formulário de busca

Method POST os dados não são enviados no endereço- deve ser usado em criação de logins e cadastros, caso contrário os dados do usuário serão exibidos no endereço do navegador

Action é o endereço para onde serão enviados o dados do form.// se estiver no mesmo domínio do site principal não precisa colocar o endereço completo.

Formulário 4

TAG 'select' também pode ser usada no formulário

TAG tipo select

Atributo size determina o tamanho da visualização das opções e o atributo multiple permite selecionar mais de uma opção, por fim o selected determina qual opção será selecionada por padrão.

Formulário 6

Novos tipos de input para formulários que vieram com HTML5

input tipo color

serve para escolher uma cor - nem todos os navegadores suportam - o input passa o código hexadecimal da cor

input tipo date

serve para escolher uma data, é possivel delimitar o período de escolha por meio do atributo max e min -- data deve ser escrita (ano-mês-dia)

input tipo datetime-local

serve para escolher uma data e a hora

input tipo file

serve para escolher enviar um arquivo - somente pode ser usado se o método do formulário for "post"

input tipo range

cria um slider para um intervalo de valores

input tipo tel

abre apenas um teclano numérico quando acessado por aparelhos mobile
